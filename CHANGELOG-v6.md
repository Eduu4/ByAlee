# ByAlee v6 — cambios principales

- Login real con Supabase Auth.
- Panel protegido y cierre de sesión.
- Reserva pública en `/` y `/reservar`.
- Datos compartidos entre dispositivos mediante PostgreSQL.
- Vercel Functions para reservas públicas.
- Validación de superposición de horarios en servidor y base de datos.
- RLS por estudio para el panel administrativo.
- Solicitudes en tiempo real.
- Servicios, clientas, fichas, citas, historial, inventario, configuración y bloqueos persistentes.
- Fotos y comprobantes en un bucket privado.
- Rutas limpias de Vercel y documentación de despliegue.
